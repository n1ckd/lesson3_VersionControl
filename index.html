What is Version control?

A version control system is a way of managing files for a project. It allows different versions or stages of a project to exist at once. These versions can be edited and changed completely from one another and not interfere with other versions of the project. Version control systems can be applied to many types of projects. Version control systems can be used to record changes to documents, websites, programs and other collections of information. The most common types of people who will use version control systems are developers or people working with code to write software and programs.

Github

Github is a website that acts as a remote repository for developers using a version control system. It is a web based hosting system that uses Git. It allows developers to manage and publish their projects. By publishing their projects it allows them to share their ideas with the community in a easy and accessible open source format. Github works to promote communities to work towards open source projects. It also allows developers to work on the same project without interfering or overwriting each others work. Github also allows developers to establish themselves on a free and popular network and showcase their work.
